# AntiAFK

Minimal Windows keep-awake app.

## Run

1. Launch `AntiAFK.exe`.
2. Press `Home` to show or hide the menu.
3. Use the toggle or your custom hotkey to start or stop.

## Features

- Custom hotkey
- Adjustable interval
- Editable key groups
- Auto-saved settings in `config.ini`
